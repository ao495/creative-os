## Quick Start

### Running Batch Processing

```bash
# Batch processing of nostall logs
python nostall/minis/creative_vector_minis/run_nostall_logs_batch.py \
    --dir nostall_logs \
    --max-files 1
```

### Generating Visualizations

```bash
# Creative universe visualization
python nostall/minis/creative_vector_minis/visualize_creative_universe.py \
    --results output/nostall_logs_batch/batch_results_*.json \
    --summary output/nostall_logs_batch/batch_summary_*.json

# 3D Creative Planet Model generation
python nostall/minis/creative_vector_minis/visualize_creative_planet_model.py \
    --pipeline-outputs output/nostall_logs_batch/pipeline_outputs_*.json
```

### Generating Analysis Reports

```bash
# Essence and rules analysis
python nostall/minis/creative_vector_minis/analyze_essence_and_rules.py

# Creative evolution theory generation
python nostall/minis/creative_vector_minis/analyze_creative_evolution.py

python nostall/minis/creative_vector_minis/analyze_creative_circuit.py
```
